私について

基本の事を大切にデザインしたいです。
周りの方との関係も大切に、巻き時計の用にコツコツと。

  • 写真を使ったデザインがすき

  • HTML CSSのコーディングが好き

趣味は旅行と食べること。
職業訓練校を卒業した後に美容部員をして、そのあと派遣でいくつかのコールセンターと事務職を経験しました。
資格をいくつか取り、IT職種（ヘルプデスクに）に転向し計１年半ほど勤務したのが前職。(2021年3月末に退職 )
(資格：パソコン検定3級、ＩＴパスポート、MSOffice Excel Specialist、MS Office Word Specialist )
ヘルプデスクでは お客様の状況に合わせた対応をすることと、情報を漏れなく聞き出して状況の切り分けを行って少しでも早く解決することに務めていました。
